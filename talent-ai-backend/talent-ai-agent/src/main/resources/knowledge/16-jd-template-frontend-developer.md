岗位名称：前端开发工程师

岗位职责：
1. 负责 Web 前端页面的开发、优化和维护。
2. 与 UI 设计师和后端工程师紧密配合，实现高质量的产品界面。
3. 封装和维护前端公共组件，提升团队开发效率。
4. 关注前端性能优化，提升页面加载速度和用户体验。
5. 研究和引入前端新技术，推动技术升级。

任职要求：
1. 本科及以上学历，3年以上前端开发经验。
2. 精通 HTML5、CSS3、JavaScript / TypeScript。
3. 熟练使用 Vue.js 或 React 框架，有实际项目经验。
4. 熟悉前端工程化工具（Vite / Webpack）和包管理工具。
5. 了解 HTTP 协议和浏览器渲染原理。
6. 有良好的代码规范和文档习惯。

加分项：
- 有 Node.js 服务端开发经验。
- 了解微前端架构。
- 有数据可视化（ECharts / D3.js）开发经验。

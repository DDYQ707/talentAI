岗位名称：Java 开发工程师

岗位职责：
1. 负责公司核心业务系统的后端设计与开发，保证系统的高可用和高性能。
2. 参与系统架构设计、技术方案评审及代码审查。
3. 根据产品需求完成功能模块的设计、编码、测试和上线。
4. 持续优化现有系统性能，排查和解决线上问题。
5. 编写和维护相关技术文档。

任职要求：
1. 本科及以上学历，计算机相关专业，3年以上 Java 开发经验。
2. 扎实的 Java 基础，熟悉 Spring Boot / Spring Cloud 微服务体系。
3. 熟悉 MySQL、Redis、消息队列等常用中间件。
4. 具备良好的系统设计能力和问题分析能力。
5. 有分布式系统开发经验者优先。
6. 良好的沟通表达能力和团队协作精神。

加分项：
- 有 SaaS 平台开发经验。
- 了解 Docker、Kubernetes 等容器化技术。
- 熟悉 AI / 大数据相关技术栈。

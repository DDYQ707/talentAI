## 3.5 前端假数据（Mock）清单 — 做什么 & 怎么做

> **原则**：后端 API 已存在的优先对接；后端没有的先建 API 再改页面。  
> **改法通用步骤**：① 新建/补充 `src/api/*.ts` → ② 页面 `onMounted` 调接口替换硬编码数组 → ③ 按钮绑定 mutation → ④ 空态/加载/错误态。

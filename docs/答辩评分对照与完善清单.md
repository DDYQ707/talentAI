## 变更影响说明（重要）

| 变更类型 | 对现有功能的影响 |
|----------|------------------|
| Sentinel + Feign Fallback | **无影响**。服务正常时行为与改前一致；仅当 `talent-job` 等下游不可用时 dashboard 返回 0/空数据而非 500 |
| Nacos Config + `@RefreshScope` | **无影响**。Nacos 无配置时仍用本地 `application.yml`；`import-check.enabled=false` 保证可正常启动 |
| 网关 401 JSON | **无影响**。前端本就对 401 做拦截；仅响应体从空变为 JSON，便于 Postman 断言 |
| 风控下架 Feign 联动 | **增强**。admin 下架时会同步暂停 `talent-job` 岗位（status=2）；Feign 失败时仍完成 admin 侧记录 |
| 公告广播推送 | **增强**。广播时会向 auth 批量创建通知；推送失败则广播事务回滚 |
| Admin 大屏 Mock（后端） | **可见变化**。无数据时行业分布/环比显示空或 0，不再返回假数据 |
| 布局用户名 | **UI 增强**。显示登录用户 nickname，无 API 变更 |
| 风控 API 路径修复 | **修复 Bug**。前端路径改为 `/api/admin/jobs/risk/{id}/takedown` |
| SQL 补丁 | **数据库变更**。Docker MySQL 已建 `talent_admin_db` 及 `job_post_id` 字段 |

**结论：核心业务链路不变，主要是架构加分项 + 两处业务联动修复。**

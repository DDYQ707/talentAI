# 智能招聘与人才画像分析系统 — 功能清单

> **文档说明**：基于仓库代码静态分析生成（前端 `talent-ai-front` + 后端 `talent-ai-backend`），不依赖已删除的历史文档。  
> **生成日期**：2026-06-28  
> **状态图例**：✅ 已完成 · 🟡 部分完成 · ❌ 未完成

---

## 一、系统概览

| 维度 | 说明 |
|------|------|
| 架构 | 微服务 + API 网关（8080）+ 单页前端（Vue 3） |
| 后端服务 | 9 个可运行服务 + 1 个公共库（`talent-common`） |
| 前端应用 | 单一 SPA，按角色路由至 HR / 管理员 / 求职者 / 面试官四端 |
| 核心业务链 | 登录 → 投递 → AI 解析/匹配 → HR 筛选 → 面试 → Offer → 人才库归档 |

| 服务 | 端口 | 状态 | 说明 |
|------|------|------|------|
| talent-gateway | 8080 | ✅ | JWT 鉴权、路由转发、CORS |

### 整体评估

- **核心业务闭环**（登录 → 投递 → AI → 面试 → Offer → 人才库）**已基本打通**，前后端均有真实 API 对接。
